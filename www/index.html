💻 Developer CMS API Reference

To connect this client-side app to real-time US health plans, you can integrate the official **CMS Marketplace API**.

View Endpoint & Integration Specs

Register and get a token at developer.cms.gov/marketplace-api.

Pasting your key initiates live API fetches instead of simulated plans.

POST https://marketplace.api.healthcare.gov/api/v1/plans/search?apikey=YOUR_KEY

Request Payload:
{
  "household": {
    "income": 48000,
    "people": [
      { "age": 35, "aptc_eligible": true }
    ]
  },
  "market": "Individual",
  "place": { "state": "MA", "zipcode": "02108" },
  "year": 2026
}
